Libinput Touchpad

Libinput es una biblioteca que proporciona una pila de entrada completa para servidores de pantalla, esta es información proporcionada por el kernel y se combina con una base de datos de hardware específica de libinput para ajustar los umbrales de cada evento generado por el dispositivo.
Luego del envío de información por medio del uso de eventos, Libinput codifica cada evento para que pueda ser procesado por el protocolo “Wayland” que proporciona un método para que los gestores de composición de ventanas se comuniquen directamente con las aplicaciones y el hardware de vídeo, siendo posible así la visualización en pantalla de cada gesto generado por el Touchpad.
